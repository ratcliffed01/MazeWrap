
Pre-Requisite
=============
Create sub folder Samples and copy program findRoute.java to it.


Directions to Compile
=====================

Change default to Samples compile from command prompt as follows :-

	 C:\...\Samples>javac -cp ../ findRoute.java

Directions to Execute
=====================
If compile is successful then execute as follows :-

	C:\...\Samples>java -cp ../ Samples.findRoute input.txt

The full path name can be used with the file name ie C:\...\Samples\input.txt

If file name not found then error message is displayed.
